前言

Python 可以通过 ctypes 模块调用 C 语言实现的函数，调用的方式很简单：

  1. 创建一个 c 文件，然后编写对应的函数
  2. 使用 c 编译器创建一个动态链接库文件（.so）
  3. 在 Python 文件中创建一个 CDLL 的实例
  4. 最后通过 {CDLL_instance}.{function_name}({function_parameters}) 格式调用 c 语言对应的函数
