发表于 2020-12-30 更新于 2022-04-24

前言

通过 Docker 安装 Zabbix Web 界面默认的语言是英文，当切换成中文的时候，有些图标显示的标签就会乱码，不能正常显示，出现这个问题的原因是在于 zabbix 默认的字体对中文支持有问题，所以解决办法就是替换默认的字体。
